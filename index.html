Research & Thesis

EyeSpeak is a mobile assistive system designed to help visually impaired pupils access printed learning materials independently. It captures printed text using the device camera, converts it into readable text, and immediately reads it aloud using text-to-speech. The system supports both English and Tagalog languages, includes replay functionality for saved readings, and provides accessibility features such as zoom, flash support, and voice feedback. It is designed to function fully offline after installation, ensuring consistent usability in classrooms and environments without internet access.
