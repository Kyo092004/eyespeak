Research & Thesis Description

EyeSpeak is an assistive mobile application developed to improve access to printed learning materials for visually impaired students. The application uses Google ML Kit for optical character recognition to detect printed text from books, worksheets, and documents through the smartphone camera. After text detection, Android Text-to-Speech converts the recognized text into spoken audio so that users can listen to the content instead of reading it visually. By combining OCR technology and speech synthesis, EyeSpeak helps visually impaired pupils access printed educational materials more independently.
